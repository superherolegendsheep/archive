Local Manager

博客管理工具

这里会生成可上传到 GitHub 的文件。浏览器无法强制选择下载文件夹，所以请把下载文件放回当前工作区或直接上传到仓库根目录。

站点信息、头像、颜色

建议保存到工作区：site.config.js

新增文章

可以直接粘贴正文，也可以选择 .html、.md、.txt、.docx 文件。文章统一作为“博文”管理，不再区分文本和 HTML 类型。

文章文件

建议保存到工作区，然后上传到 GitHub 仓库根目录。HTML 文件也会作为独立页面供读者直接打开。

新的 posts.json

建议保存到工作区：posts.json

管理文章

删除会从目录中移除文章记录。仅自己可见草稿不会在首页显示，但公开仓库里的文件本身不是真正加密。

管理标签

可以统一重命名标签，或把某个标签从所有文章里移除。

上传 GitHub

这是可选功能。需要 GitHub fine-grained token，权限选择本仓库 Contents: Read and write。token 只在当前页面使用，不会写进文件。
