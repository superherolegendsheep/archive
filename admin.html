博客管理工具

这里会生成可上传到 GitHub 的文件。浏览器不能强制指定下载目录，下载后请移动到工作区或直接上传到仓库根目录。

站点信息、头像、颜色、评论接收

管理作品集

可新增、改名、删除作品集，并为每个作品集上传封面。删除作品集只会从配置中移除，不会删除文章。

新增文章

可以直接粘贴正文，也可以选择 .html、.md、.txt、.docx 文件。文章统一作为博文管理。

文章文件
